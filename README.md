# Dario-Hackpad

Inspiration:

I created this hackpad to learn the fundamentals of electronics and product design. I really enjoy making things from scratch, and this project gave me hands-on experience with the full design process. I hope to work on another blueprint project in the near future.

Challenges:

It was my first time using KiCad and it took me a very long time to set up and understand the controls and keyboard shortcuts. It took me 1 hour to understand how to add the symbols and footprints to the program, it took me a lot of google searches and prompts from chatgpt to help me setup the KiCad.

BOM:

* 6x Cherry MX Switches
* 2x SK6812 MINI Leds
* 1x XIAO RP2040
* 4x Blank DSA Keycaps
* 4x M3x16 Bolt
* 4x M3 Heatset

Others:
* KMK Firmware
* Top Case.step
* Dario Hackpad.step
